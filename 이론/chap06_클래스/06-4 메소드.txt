[핵심 키워드]
- 선언부: 메서드 선언부는 리턴 타입, 메서드 이름, 매개 변수 선언 부분을 말함
- void: 리턴값이 없는 메서드는 리턴 타입으로 void를 기술해야함
- 매개변수: 메서드 호출 시 제공되는 매개값은 메서드 선언부의 매개 변수에 차례대로 대입되어, 메서드 블록 실핼시 이용됨
- 리턴문: 메서드 호출 시 제공되는 매개값은 메서드 선언부의 매개 변수에 차례대로 대입되어야함. 리턴 타입이 void라면 return문은 필요없지만, 메서드 실행 종료를 위해 사용할 수도 있음
- 호출 : 메서드 이름(매개값...) 형태로 호출함
- 오버로딩 : 클래스 내에 같은 이름의메서드를 여러개 선언하는 것을 메서드 오버로딩이라고함. 메서드 오버로딩의 조서은 매개 변수의 타입, 개수, 순서 중 하나가 달라야함
